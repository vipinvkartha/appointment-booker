# Appointment booker

This is a Node.js backend application with a PostgreSQL database, managed using Docker and Docker Compose.

## Getting Started

### Prerequisites
- Docker
- Docker Compose

### Installation
```sh
git clone https://github.com/vipinvkartha/appointment-booker
cd appointment-booker
docker-compose up --build
```

### Accessing Services
```sh
Backend API: http://localhost:3001
Adminer: http://localhost:5050
```

### Stopping services
docker-compose down
